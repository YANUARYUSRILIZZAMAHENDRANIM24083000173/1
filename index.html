Panduan Pemasangan Komputer

Step-by-step lengkap untuk merakit PC Anda

Komponen Utama

Processor (CPU)

Otak komputer yang melakukan pemrosesan data

RAM

Memori sementara untuk menyimpan data yang sedang diproses

GPU

Memproses dan menampilkan grafis pada layar

Motherboard

Papan utama tempat semua komponen terhubung

Langkah-Langkah Pemasangan

1

Persiapan Motherboard

Yang Perlu Dilakukan:

  • Siapkan motherboard pada permukaan yang bersih dan rata
  • Pastikan Anda memiliki manual motherboard
  • Siapkan sekrup dan standoff untuk casing

Perhatian!

Sentuh grounding (logam) casing sebelum memegang motherboard untuk menghindari ESD.

2

Pemasangan CPU

Perhatikan tanda segitiga pada CPU dan socket untuk penyesuaian

Prosedur:

  1. Buka tuas pengunci socket CPU
  2. Letakkan CPU dengan hati-hati sesuai tanda orientasi
  3. Tutup kembali tuas pengunci dengan lembut
  4. Oleskan thermal paste jika perlu
3

Pemasangan RAM

Petunjuk:

  • Buka pengunci di kedua ujung slot RAM
  • Sesuaikan takik pada RAM dengan slot motherboard
  • Tekan dengan merata hingga pengunci menutup sendiri
  • Untuk dual channel, gunakan slot dengan warna sama
Lihat Langkah Selanjutnya (4-8)
4

Pemasangan GPU

Pasang GPU ke slot PCIe x16, pastikan pengunci slot terbuka sebelum memasang. Sekrup bracket GPU ke casing untuk stabilitas.

5

Kabel Power

Hubungkan kabel power dari PSU ke motherboard (24-pin ATX dan 8-pin CPU), GPU, dan komponen lain yang membutuhkan daya.

6

Pemasangan Storage

Prosedur:

  1. Untuk M.2 SSD: Pasang dengan sudut 30° ke slot M.2 lalu tekan dan kencangkan sekrup
  2. Untuk SATA SSD/HDD: Pasang di drive bay dan hubungkan kabel SATA dan power
  3. Pastikan koneksi yang rapat dan kuat
7

Manajemen Kabel

Tips:

  • Gunakan cable tie untuk mengikat kabel yang berdekatan
  • Manfaatkan rongga belakang motherboard tray untuk menyembunyikan kabel
  • Pastikan kabel tidak menghalangi airflow casing
8

First Boot & BIOS Setup

Langkah:

  1. Hubungkan kabel power dan nyalakan PC
  2. Masuk BIOS dengan menekan tombol DEL/F2 saat boot
  3. Verifikasi semua komponen terdeteksi
  4. Atur boot priority jika perlu install OS
  5. Simpan perubahan dan exit

Troubleshooting:

Jika PC tidak menyala, periksa kembali koneksi power dan pemasangan RAM/GPU
